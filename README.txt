MC BarberShop

Este sitio web es una landing page para una barbería profesional que ofrece servicios de estilismo, cursos, productos para el cuidado personal y presencia en redes sociales. Incluye secciones de inicio, redes, conócenos, galería, cursos y productos, con navegación suave, contador de seguidores animado y un filtro dinámico para la tienda de productos.
